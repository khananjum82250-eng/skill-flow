# SkillFlow

SkillFlow is a Flask-based skill exchange web app with user and admin dashboards.

## Active Structure

- `app.py` - Flask application entry point
- `templates/user` - active user templates
- `templates/admin` - active admin templates
- `templates/partials` - shared template partials
- `static/css` - active stylesheets
- `static/js` - active scripts
- `static/images` - active image assets and avatars
- `static/uploads` - runtime uploads
- `database` - database schema and maintenance SQL
- `logs` - local Flask runtime logs
- `tests` - project tests
- `backup_final_safe` - archived old backups and unused files
